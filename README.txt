Enzuzo Cookie Consent is a cookie consent manager that builds trust and keeps you compliant.

== Description ==

Empower your audience to take charge of their consent, preferences, and first-party data through a user-friendly, all-in-one portal. Comply with global privacy laws like GDPR & CCPA.  See https://www.enzuzo.com/consent-management-software for more information.

== Installation ==

To install the plugin:

1. Upload the content of the plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in Wordpress
1. Specify the account UUID (looks like aaaaaaaa-bbbb-cccc-dddd-eeeeeeeeeeee) and enable the banner in Wordpress under Settings -> Enzuzo

== Frequently Asked Questions ==

= Do I need an account to use this plugin? =

Yes.  An account is required to customize the plugin and be able to store and access consent logs, etc.

= Where do I find the UUID =

Once an account is created, you can find the UUID under this URL https://app.enzuzo.com/settings/cookiebar/add-to-website
